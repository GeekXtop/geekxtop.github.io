利用Github实现免费翻墙

由于众所周知的原因，以我自己的经历为例写成教程，希望可以帮助各位自由冲浪。

前言

本文的适用人群

声明：本文仅作为技术分享，使用者请务必遵守当地法律法规。

你可能会说，我知道某某软件可以，为什么还需要看一篇这么长的教程？说明一下我个人非常反对使用封装好的一键翻墙软件或 App，第一是节点质量无法保证，且使用人数众多（这也是造成一些账户封号的主要原因），第二，非开源客户端可能存在安全隐患或后门，尤其是 Android 和 Windows；第三，付费跑路风险非常之高。

首先确认你的网络环境是否可以正常访问 github.com以及下载Github中的文件，经过实测几个翻墙软件的项目库地址都被屏蔽了，在未翻墙状态下无法访问，请参阅：

GitHub访问慢打不开？快使用加速神器FastGithub

寻找免费节点

在 GitHub 中搜索关键字如“免费 节点”，右上角可以选择不同的排序方式筛选结果，推荐选择 Recently updated（最近更新） 可以筛选出活跃项目，这些都是收集了全网免费节点资源的项目。

比如我挑选出来如下几个我测试时可正常使用的，如失效请请自行寻找。

  • https://github.com/freefq/free
  • https://github.com/yu-steven/openit
  • https://github.com/aiboboxx/v2rayfree
  • https://github.com/alanbobs999/TopFreeProxies

打开其中一个项目会看到类似上图这样的订阅链接，复制订阅链接，页面内容可能会有不一致请自行查看描述。

对于页面内的付费和引流内容请不要轻易相信。

免费节点订阅链接可能存在安全风险，请注意个人信息安全，使用完及时关闭。

这些节点适合使用频率较低，稳定性要求不太高的人群，比如学生党查文献，查资料等。

Base64 可供 v2Ray 使用，Clash 都是专用链接。接下来教你各个客户端如何使用。

冲浪客户端下载和使用

目前主流的客户端有 v2Ray 和 Clash 两种，v2Ray 客户端无法使用 Clash 节点订阅，Clash 要使用 v2Ray 订阅也需要进行转换，不适合新手操作，所以我们直接使用现成的 订阅链接 就好，v2Ray 和 Clash 两者选其一就可以了。

请根据你所使用的设备类型选择客户端，跳转至对应章节查看简明教程。

iOS

iOS 因为政策原因冲浪工具类软件无法在国区 App Store 上架。所以你需要先注册一个 海外Apple ID 。如何注册请参阅：

iOS 注册海外 Apple ID 最新教程

由于注册过程中可能先需要 科学冲浪 环境，所以容易陷入一个死循环，因为不能科学冲浪所以需要

如果你想白嫖到底，那么我推荐你使用 v2Ray 客户端 OneClick，它是 免费 的，但体验一般。

如果你想使用 Shadowrocket，售价 2.99 美元 （美区）。请自行在 某宝 搜索关键词 iOS 礼品卡 进行充值后购买。

3.1 v2Ray

v2RayN Windows

https://github.com/2dust/v2rayN/releases

请选择v2rayN-Core.zip，即带有完整 v2Ray 核心的包。

解压后打开文件夹运行v2rayN.exe，默认最小化，请在任务栏找到该图标双击打开主界面。

添加订阅和排序节点

  • 点击 订阅 - 订阅设置 - 添加
  • 地址 url 填写上一章节获取到的订阅链接，填写 备注
  • 确保勾选 启用 再 确定
  • 点击 订阅 - 更新订阅 即可刷新节点信息。

介绍几个常用的快捷键：

  • Ctrl + A 全选节点
  • Ctrl + R 测试选中节点真连接延迟 （ping Google）
  • Enter 回车选择节点（最左侧有 √ 且呈该条目呈蓝色为当前选中节点）

点击 测试结果 表头进行升序排序，再选择延迟较低的节点。

更新订阅并选择低延迟节点，这是每次打开后都需要做的步骤。

配置和启用

右键任务栏图标，点击 系统代理 - 自动配置系统代理，图标变为红色。此时已经处于冲浪状态。

路由 一般情况下默认 绕过大陆 即可。

使用完或关机前请记得 清除系统代理 ，否则可能导致无法上网。

v2RayNG Android

https://github.com/2dust/v2rayNG/releases

请选择v2rayNG_x.x.x.apk。

或 Google Play 下载：

https://play.google.com/store/apps/details?id=com.v2ray.ang

下载安装后打开 v2rayNG。

添加订阅和排序节点

主界面点击左上角，选择 订阅设置 。

再点击右上角 + 号，粘贴上一节获取的 订阅链接 ，点击 √ 保存。

回到主界面，点击右上角 三个点，依次选择：

  • 更新订阅
  • 测试全部配置真连接（ping Google）
  • 按测试结果排序

更新订阅并选择低延迟节点，这是每次打开后都需要做的步骤。

配置

主界面点击左上角，选择 设置 。有 2 处建议配置。

  • 点击 分应用代理 ，打开 分应用代理 的开关 ，再点击右上角三个点，选择 自动选中需代理应用。其他应用根据实际情况勾选。

  • 预定义规则 选择 绕过局域网及大陆地址而后代理 。

启用

点击 主界面 V 图标 启动代理。

Qv2ray MacOS/Windows/Linux

https://github.com/Qv2ray/Qv2ray/releases

  • MacOS 请选择Qv2ray-x.x.x-macOS-x64.dmg
  • Windows 请选择Qv2ray-v2.7.0-Windows-Installer.exe安装或使用便携版Qv2ray-v2.7.0-Windows.7z
  • Linux 请选择Qv2ray-v2.7.0-linux-x64.AppImage

OneClick 免费

下载运行

https://apps.apple.com/us/app/oneclick-safe-easy-fast/id1545555197?l=zh

添加订阅和排序节点

  • 点击右上角 + 号，选择 添加订阅。

  • 地址 输入 订阅链接 ，点击 √号 保存。

请参阅：

保存订阅时提示“no base64 data”怎么解决

配置和启用

点击 左上角 齿轮 进入设置。

  • 服务器 可以管理节点，更新订阅，进行测速等。
  • 订阅 可管理订阅链接。
  • 代理模式 有 自动、全局、直连 三种，一般自动 即可。
  • 强制代理 可添加网址强制走代理。
  • 强制直连 可添加网址强制不走代理。

主界面 可以选择节点， 点击 下方按钮 即可启用 VPN 连接。

FAQ

保存订阅时提示“no base64 data”怎么解决

  1. 最快的方法，换一个 订阅链接 。

  2. 这种方法需要每次复制导入较为麻烦。

    • 全选页面下方节点的文本，复制。

    • 主界面点击右上角 + 号，选择 添加配置，选择 从剪贴板导入 。

3.2 Clash

Clash for Windows (CFW) Windows/MacOS/Linux

https://github.com/Fndroid/clash_for_windows_pkg/releases

  • Windows 请选择Clash.for.Windows.Setup.xx.exe安装或便携版Clash.for.Windows.Setup.xx-win.7z；
  • 32 位 Windows 请选择Clash.for.Windows.Setup.xx.ia32.exe；
  • MacOS 请选择Clash.for.Windows-xx.dmg；
  • Linux 请选择Clash.for.Windows-xx-x64-linux.tar.gz；
    下载并安装。

汉化

汉化补丁下载

https://github.com/BoyceLig/Clash_Chinese_Patch/releases

我之前下载的 CFW 版本是Clash.for.Windows.Setup.0.19.14.exe,所以下载 Clash for Windows v0.19.14 的汉化补丁，点击 app.7z。

注意：一定要确保汉化补丁和主程序的版本一致，避免出现问题，后续 CFW 版本升级后也要升级汉化补丁，替换时请确认 CFW 已关闭

汉化补丁安装
  • CFW 快捷方式右键，选择打开文件所在的位置

  • 解压app.7z得到app.asar，拖入resources路径中替换原本的 app.asar 文件即汉化成功。重新打开 CFW 可以查看效果。

导入配置并启用代理

  • 进入 配置 选项卡，输入上面得到的 Clash 订阅链接，点击 下载，下方出现配置文件。

  • 进入常规选项卡，打开 系统代理 选项，即可启用代理。

任务栏图标如图所示，此时电脑已经处于冲浪状态。

注意：关机前最好先关闭系统代理，否则可能会出现无法正常上网的情况

MacOS，Linux 同理不再详述。

FAQ

  • Windows 关闭代理后电脑无法正常上网

可能是关机前没有正常关闭代理软件导致。请进入 Windows设置 - 网络和Internet - 代理 - 手动设置代理 - 关闭 使用代理服务器 选项。
