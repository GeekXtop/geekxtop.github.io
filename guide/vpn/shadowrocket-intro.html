iOS 科学上网神器 Shadowrocket 新手教程

Shadowrocket 是一款 iOS / iPadOS 平台上非常强大的代理客户端，对于 v2Ray，Clash 等主流协议全面支持，俗称“小火箭”。

请参阅：
科学上网如何挑选付费“机场”

下载

首先你需要一个海外 Apple ID 并且余额大于 $2.99 ，如何注册海外 Apple ID 和充值礼品卡请参阅：

iOS 注册海外 Apple ID 最新教程

请认准以下链接。

https://apps.apple.com/us/app/shadowrocket/id932747118

注意： Shadowrocket 没有在国区上架。不是 “Shadow Rocket”，不是 “Shadowrock”，不是 shadowrocket VPN，也不是 “ShadowRocket 网络助手”等等，这些全部是李鬼，千万不要下载！

添加订阅

运行 Shadowrocket ，点击右上角 + 按键。

  • 类型 选择 Subscribe

  • URL 输入 订阅链接

  • 点击 完成 即可添加订阅。

设置

进入 设置 页

延迟测试方法

推荐选择 CONNECT ，即测试真连接，准确反映节点是否可用。

订阅

进入 设置 - 订阅 。

推荐开启 打开时更新 和 自动后台更新。

开启 根据 Ping 排序，根据连通性测试结果排序。

测试延迟并启用

回到 首页 ，点击 连通性测试。此时会测试节点延迟并根据 Ping 值排序，选择下方排名靠前的节点。

右测开关即为 启用，此时已处于科学上网状态。

全局路由 有几个选项。 配置 为配置文件代理，即按照规则自动分流，只有特定网站通过代理， 代理 为全局代理，即所有连接均通过代理； 直连 即所有连接均不通过代理， 场景 适用于不同网络环境下自动切换代理模式。

推荐选择 配置 ，请继续查看下一章节。

配置文件

推荐使用以下项目的配置文件。该项目是由 @Hackl0us 搜集整理的一套适用于 Shadowrocket 的精简、实用、高效的规则。

https://github.com/Hackl0us/SS-Rule-Snippet

点击下方菜单栏的 配置 ，并选择 添加配置 。在弹出的窗口中，粘贴规则链接：

https://fastly.jsdelivr.net/gh/Hackl0us/SS-Rule-Snippet@master/LAZY_RULES/Shadowrocket.conf

点击 下载 按钮。

添加完成后，可在 远程文件 中看到刚刚粘贴的链接。点击该链接，在弹出的菜单中选择 使用配置 。

稍等片刻可以发现 本地文件 中出现 ShadowRocket.conf 的配置文件，右侧有 ✔️ ，表示已经启用该配置文件。
